Privacy policy

The app Bloomable Logistics does not collect or share any information for commercial purposes. All user data are only used for optimizing the app experience itself.

The app will collect error/crash reports in the background. The information in these reports are only used for analyzing the error and will be used in an attempt to fix the error in the code. These reports may include details of the order if the error comes from an action applied to that order. The error reports will include a value that corresponds to the client's device ID. This value is only used for the purpose of grouping error logs together. The information in these logs are not shared with any party, except from the logging framework itself.
Error logs are automatically deleted after 30 days.

The app does not collect the Bloomable username or password information which is used to log in to the Bloomable servers (bloomable.co.za). The password is not stored on the device or external servers other than bloomable.co.za. A modified version of the username is securely stored on the device and used for the push notification service, if enabled so in the app settings. This value is not shared with any parties, except for Firebase Cloud Messaging (FCM). FCM will keep this information private and secure so that even we cannot access this.

The user cannot request to remove its data:

  • The error/crash logs will automatically be removed after 30 days.
  • For removal of the data on the bloomable.co.za, the user should contact them directly.
  • The stored data on the device will be removed if the user uninstalls the app.
